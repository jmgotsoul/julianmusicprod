Concept Notes

title note
PSYTECH This is primarily a piano concept with a few other things to add to the dissonance. I thought about some of the interaction Michael has with PSYTECH. I also thought about the montage of images on page 3 of the script.
Solution Q This is kind of a uneasy feeling cue. For some reason I thought of the EXT. DOWNTOWN - NIGHT scene and then the INT. MICHAEL'S OFFICE scene on page 3 so I named it Solution Q. I also thought of the INT. EVERGREEN CONFERENCE ROOM - DAY scene that starts on page 2.
Loss This one could possibly be used in more somber and sad moments, possibly when Michael is reflecting on his family.
Serum C This is another approach to some buildup music approaching the last scene. It does have piano and strings but it also has some grungy/gritty guitar and percussion as well.
